# Catherine Kamau — Data Analyst Portfolio
## 🚀 Deploy to Vercel in 2 Minutes

### Step 1 — Push to GitHub
1. Create a new GitHub repo called `portfolio`
2. Upload both `index.html` and `vercel.json`

### Step 2 — Deploy on Vercel (free)
1. Go to https://vercel.com → sign up with GitHub
2. Click **Add New Project** → select your `portfolio` repo
3. Click **Deploy** — Vercel gives you a free URL like `catherine-kamau.vercel.app`

Every time you push a change to GitHub, Vercel auto-redeploys. ✅
